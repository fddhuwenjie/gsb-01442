# 组件 API

Alkaid Plus 封装了 Element Plus 的所有组件，API 与原组件保持一致。

| AkLink | 链接 | ElLink |

## API 文档

所有组件的 API（Props、Events、Slots、Methods）与 Element Plus 完全一致。

请参考 [Element Plus 官方文档](https://element-plus.org/zh-CN/component/button.html) 获取详细的 API 说明。
